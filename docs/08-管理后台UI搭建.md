# 08 - 管理后台 UI 搭建

## 本节目标

- 安装 shadcn/ui 组件库
- 搭建侧边栏 + 顶栏布局
- 实现登录页面
- 实现用户管理列表页（搜索、状态切换、删除）

---

## 安装 shadcn/ui

```bash
pnpm dlx shadcn@latest init
# 选择：Radix → Nova 预设

pnpm dlx shadcn@latest add button input label card table badge avatar dropdown-menu sheet sidebar tooltip separator
```

---

## 路由结构

```
src/app/
├── (auth)/                   ← 登录相关（无侧边栏）
│   ├── layout.tsx
│   └── login/
│       └── page.tsx          →  /login
└── (dashboard)/              ← 后台页面（有侧边栏）
    ├── layout.tsx
    ├── dashboard/
    │   └── page.tsx          →  /dashboard
    └── users/
        └── page.tsx          →  /users
```

> `(auth)` 和 `(dashboard)` 是路由分组，括号里的名字不出现在 URL 中

---

## 关键文件

### 后台布局 `src/app/(dashboard)/layout.tsx`

使用 shadcn SidebarProvider 包裹，包含侧边栏 + 顶栏

### 侧边栏 `src/components/app-sidebar.tsx`

菜单项：仪表盘 / 用户管理 / 角色管理 / 部门管理

### 登录页 `src/app/(auth)/login/page.tsx`

- 账号密码表单
- GitHub 登录按钮
- 错误提示
- 登录成功跳转 `/dashboard`

### 用户管理页 `src/app/(dashboard)/users/page.tsx`

功能清单：
- ✅ 用户列表（Table 展示）
- ✅ 搜索（关键词过滤）
- ✅ 分页
- ✅ 状态切换（点击 Badge 直接切换）
- ✅ 删除用户（带确认弹窗）
- ✅ 头像展示（AvatarFallback 兜底）
- 🔜 新增用户弹窗
- 🔜 编辑用户弹窗

## 状态切换技巧

```tsx
// 点击 Badge 直接切换状态，无需弹窗
<Badge
  variant={user.status ? 'default' : 'destructive'}
  className="cursor-pointer"
  onClick={() => handleToggleStatus(user)}
>
  {user.status ? '正常' : '禁用'}
</Badge>
```
