- 编辑时回填数据，邮箱不可修改
- 角色通过 Checkbox 多选
- 提交后刷新列表

```tsx
// 新增 vs 编辑 判断
const isEdit = !!user

// 编辑时只传 name + roleIds，不传 email/password
const body = isEdit
  ? { name: form.name, roleIds: form.roleIds }
  : { email: form.email, password: form.password, name: form.name, roleIds: form.roleIds }
```

### 权限按 module 分组展示

```typescript
// 把平铺的权限按 module 分组
const permByModule = permissions.reduce((acc, p) => {
  if (!acc[p.module]) acc[p.module] = []
  acc[p.module].push(p)
  return acc
}, {} as Record<string, Permission[]>)
```

### 树形权限展开为平铺列表

```typescript
function flatten(items: any[]): any[] {
  return items.flatMap((item: any) => [item, ...flatten(item.children || [])])
}
```

### 权限超出时缩略展示

```tsx
{role.rolePermissions.slice(0, 3).map(rp => (
  <Badge key={rp.permission.id} variant="outline">{rp.permission.name}</Badge>
))}
{role.rolePermissions.length > 3 && (
  <Badge variant="outline">+{role.rolePermissions.length - 3}</Badge>
)}
```
